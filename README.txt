To run these programs, you must install all the packages used such as: ultralytics
and everything in requirements.txt

For yolo to work, you need to download the zip file from: https://github.com/ultralytics/ultralytics
and place into C: drive

For new data classes to work, you must use the label tool from https://github.com/ivangrov/ModifiedOpenLabelling
and insert your new images in the images folder

For the AI to work on webcam, you need to train the AI using yolov8s.pt for at least 50 epochs for best results

For dragon.py to work, the image folder must be in the same directory as the game code
